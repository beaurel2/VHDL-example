# VHDL

This repository contains VHDL examples for the bachelor-level course [Design of Digital Systems](https://www.tha.de/homes/kiefer/), using [ULX3S](https://www.mouser.com/ProductDetail/Radiona/CS-ULX3S-01?qs=hWgE7mdIu5Sz6R%252BJENFe0A%3D%3D) FPGA Board and [Digital Design](https://www.hs-augsburg.de/~beckmanf/dokuwiki/doku.php), using [altera_de1](https://www.terasic.com.tw/cgi-bin/page/archive.pl?No=83) FPGA Board at Technical University of Applied Sciences Augsburg
